# Meteograma

Projeto para gerar meteogramas usando dados do GFS.

## Estrutura do Projeto

- `download_dados.py`: Script para baixar e processar dados do GFS.
- `meteogram.html`: Visualização do meteograma.
- `tools/`: Ferramentas auxiliares.
